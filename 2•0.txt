자신의 고향식으로 말해도 다른 사용자와 호환되는 언어
한국인 화자: Humaio-joi pifi-otika lokus. (SOV)

영어 화자: Humaio-joi lokus pifi-otika. (SVO)

몽골어 화자: Pifi-otika Humaio-joi lokus. (OVS)

라틴어 감성 화자: Lokus humaio-joi pifi-otika. (VSO)

→ 전부 똑같이 “사람이 불을 본다.”

humaio-joi fitao : 사람이 죽다
humaio-joi pai-fitao : 사람이 죽었다
humaio-joi fu-fitao : 사람이 죽을 것이다.

Nuki humaio-joi pai-fitao :  사람이 죽지 않았다
Whispi humaio-joi pai-fitao :  사람이 죽지 않았습니까?

(Nuki가 맨 앞에 붙으면 전체를 부정한다)
(단어-n 이라면 해당 단어만 부정한다)

응용 /
Relio, framiot! → 안녕, 친구야!

Nuki tio-otika jowet joious-siyon. → 너희들은 집에 가지 않았다.

Delio, gio-joi futroyot lokus. → 잘 가, 우리는 내일 보자.
