### **18. 조합 규칙**

* [큰 수 + 작은 수] 순서로 읽음.
* 덧셈 방식: 중국어/한국어 계열.
* 예시:

  * 11 = sol-tung
  * 12 = sol-tog
  * 20 = tog-sol
  * 23 = tog-sol-diam
  * 100 = pok
  * 305 = diam-pok-kiri
  * 1999 = tung-pok-pof-twio-pof-sol-pof
* 괄호로 묶기: 11 = (pofio tung)
